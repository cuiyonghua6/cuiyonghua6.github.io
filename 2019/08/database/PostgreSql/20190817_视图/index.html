视图概述

1、视图是一个虚拟表，是从数据库中一个或多个表中导出来的表。

2、视图的作用：

  1. 简单化，经常使用的查询可以被定义为视图，用户不必为以后的操作每次指定全部的条件；
  2. 安全性，通过视图，用户只能查询和修改他们能见到的数据；
  3. 逻辑数据独立性，视图可帮助用户屏蔽真实表结构变化带来的影响。
